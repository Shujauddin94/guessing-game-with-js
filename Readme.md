# 🎮 JavaScript Guessing Game

A fun and simple **number guessing game** built using **HTML, CSS, and JavaScript**.  
The goal is to guess the secret number generated by the game — with hints to guide you!

---

## 🚀 Features
- 🔢 Random number generation
- 💬 Real-time feedback ("Too high", "Too low", "Correct!")
- 🧮 Score tracking
- ♻️ Reset / Play again button
- 📱 Responsive design
- 🚫 Input validation

---

## 🛠️ Tech Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**

---

## 📂 Project Structure
```
/guessing-game
│── index.htm

└── script.js
```

---

## ▶️ Run Locally
1. Open the project folder in your browser, or use a simple local server.
2. If you use VS Code, you can run **Live Server** on [index.htm](index.htm).
3. The game will work once the files are served from the project directory.

---

## ⚡ Quick Start

1. Open the project folder `guessing-game-with-js` in your editor or file manager.
2. Open [index.htm](index.htm) in your browser or start a local server (e.g., Live Server in VS Code).
3. Play immediately — no build step required.

## ⌨️ Keyboard Shortcuts

- **Enter**: Check your guess
- **Esc**: Reset / start again
- **R**: Restart current round
- **C**: Clear the guess input

---

## 🎯 How to Play
1. Enter a number within the allowed range.
2. Click **Check**.
3. Read the hints and adjust your guess.
4. Keep guessing until you find the secret number!
5. Click **Play Again** to restart.
